# Development Goals

## Short-Term Goals (Next 1-2 Releases)
- Improve metadata extraction reliability for YouTube links when anti-bot checks are triggered.
- Add clearer in-app guidance for `COOKIES_FROM_BROWSER`, `COOKIES_FILE`, and JS runtime settings.
- Improve startup and link-scanning performance for large channel/profile lists.
- Add structured logging levels in the UI for easier troubleshooting.

## Mid-Term Goals (Next 3-6 Releases)
- Add download analytics (success/error counts, average processing time, per-platform stats).
- Extend platform-specific handling for changing extractor behaviors.
- Add optional batch presets for repeated export settings.
- Improve output quality controls and export profile management.
