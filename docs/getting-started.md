# Getting Started

## Prerequisites
- Python 3.10+
- ffmpeg in `PATH`
- Optional: Node.js runtime for `yt-dlp` JavaScript execution

## Setup
```bash
git clone https://github.com/VoxHash-Technologies/YTSDownloader.git
cd YTSDownloader
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## First Run
```bash
./.venv/bin/python main.py
```

Open the app, set an output folder, provide a channel/profile URL, and run one download as a smoke test.
