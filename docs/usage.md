# Usage

## Start the Application
```bash
./.venv/bin/python main.py
```

## Basic Flow
1. Select output folder.
2. Enter a source URL (YouTube/Instagram/Facebook/TikTok).
3. Choose output format and optional watermark image.
4. Start processing and monitor progress bars and logs.

## Output
- Media files are written to the selected output directory.
- Original titles and source links are recorded in `original_names.txt`.
